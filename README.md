# vant_mobile
这是一个基于Vue2.7 + Vant + less + viewPort自适应的移动端项目模版
## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Lints and fixes files
```
pnpm run lint
```
